Step 1

Infinispanを取得

ユーザー名とパスワードを入力し、以下のいずれかのコマンドを実行してください: 


podman run -it -p 11222:11222 -e USER="username" -e PASS="password" --net=host quay.io/infinispan/server:{{site.data.ispn.stable.minor_version}}
        


docker run -it -p 11222:11222 -e USER="username" -e PASS="password" quay.io/infinispan/server:{{site.data.ispn.stable.minor_version}}

Step 2

コンソールを開く

ブラウザで localhost:11222 を開き、ログインします。

Step 3

キャッシュを作成

キャッシュに名前をつけて、テンプレートからキャッシュを作成します。

Step 4

データを追加

  1. キャッシュを選択し、*Add entry（要素を追加）*を押します。
  2. キーとして "hello"、バリューとして "world" を指定します。
  3. オプションで、生存時間または最大アイドル時間を指定します。
  4. 要素をキャッシュに追加するには、*Add（追加）* を選択します。

データを追加した後にキーを入力すると、その要素を取り出して表示できます。

これで完了です。これでInfinispanを使えます。

もし、ベアメタル上で動かしたい場合は、Infinispanサーバを始めようを訪れてください。自分のアプリケーションで実行したい場合は組込みInfinispanへ訪れてください。
