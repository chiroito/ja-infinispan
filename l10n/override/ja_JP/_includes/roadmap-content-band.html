Infinispanはタイムボックス型のリリーススケジュールで、3ヶ月ごとに新しいリリースが行われます。以下のリストは、次の数回のリリースに向けた機能を示しています。サイクルの終了時に十分な品質が得られなかった場合、機能が将来のリリースに延期される可能性があることを覚えておいてください。

ご注意 以下のリストは、いつでも予告なしに変更される可能性があります。

API

  • 新しいモジュラAPI

中核

  • 衝突を解決するアクティブ-アクティブ非同期サイト横断
  • パフォーマンスとリソースの継続的な改善
  • その場でのローリングアップグレード

問い合せ

  • リモート問い合せのためProtoBuf 3をサポート

セキュリティ

  • 暗号化されたパスワードのストレージ
  • SCRAM認証機構

サーバ / リモート

  • 軽量サーバー
  • 新しいRESTful API

管理 / Monitoring

  • コンソール改善
  • CLI改善
  • サポート性改善
  • 追跡性
